# Task 4: General Health Query Chatbot

A chatbot that answers general health questions using the Groq API and Llama 3.1 model.

## Tools Used
- Python
- Groq API (Llama 3.1 8B Instant)
- Prompt Engineering
- Safety Filtering

## How to Run
1. Get a free API key from [console.groq.com](https://console.groq.com)
2. Paste your key in Cell 3 where it says `your_groq_api_key_here`
3. Run all cells in order
4. Type your health question when prompted

## Features
- Friendly medical assistant persona (MediBot)
- Safety filter blocks harmful queries
- Powered by Llama 3.1 via Groq API
