# Task 5: Mental Health Support Chatbot (Fine-Tuned)

## Overview
Fine-tuned DistilGPT2 on mental health counseling conversations to respond empathetically.

## Tools
- Python, Hugging Face Transformers, Google Colab

## Dataset
- Mental Health Counseling Conversations by Amod (3,512 samples, 70/15/15 split)

## Results
- Training Loss: 2.56 → 1.97
- Test Loss: 2.087

## How to Run
1. Open in Google Colab
2. Enable T4 GPU
3. Run all cells in order

## Note
Built for learning purposes. A larger model like Llama 3 or Mistral 7B
would produce significantly better results.
